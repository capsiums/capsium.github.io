About this capsule

This is the second page of capsium-demo — reached through routes.json, resolved by the service worker, bytes lifted out of the zip you installed.

What is inside

  • metadata.jsonname, version, guid, uuid, author Ribose, license MIT
  • manifest.jsonevery resource with its MIME type — generated by the packer
  • routes.jsonthe URL contract — auto-generated: pages, assets, datasets
  • storage.jsonone dataset, demo, backed by data/demo.json
  • security.jsonSHA-256 checksums over every file — verified at install time

Why it works everywhere

The package only declares the contract. Any Capsium reactor — the Ruby gem, the OpenResty/Lua reactor, or this service worker — reads the same files and serves the same bytes. This page made no network request to a server: the fetch was intercepted inside your browser.
